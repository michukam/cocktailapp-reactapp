- CSS

<!-- GETTING STARTED -->

## Getting Started

To use my project and run it on your computer you must follow the steps below.

### Installation

1. Install [Node.js](https://nodejs.org/en/)
2. Download the repo
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run NPM
   ```sh
   npm start
   ```
